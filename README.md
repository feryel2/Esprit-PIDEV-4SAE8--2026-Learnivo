#learnivo _ English School Management Application
# Overview

This project was developed as part of the PIDEV – 3rd Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).

Learnivo is a microservices-based e-learning platform that allows students to access courses and quizzes online.
## Features
- user Management
- Course Management
- Quiz Management
-Events Management
- Clubs Management

# Tech Stack

### Frontend
- Angular

### Backend
- Spring Boot
- MySQL
- JPA / Hibernate
